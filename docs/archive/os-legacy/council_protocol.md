# Council Mode Protocol

**Trigger**: Critical decisions (Pricing, Roadmap, Major UX, Architecture, Compliance-Sensitive Copy).

## Workflow
1.  **Call to Council**: Orchestrator invokes Council Mode with a Topic.
2.  **Proposal Phase**: 3-9 Agents produce competing proposals (Markdown artifacts with evidence/tradeoffs).
3.  **Deliberation**: Agents critique proposals (Red Team view).
4.  **Decision**: Orchestrator selects or synthesizes the winning proposal.
5.  **Record**: Decision recorded in `/docs/decisions/YYYY-MM-DD_Topic.md` with evidence links.

## Participants
- Executive Orchestrator (Chair)
- Relevant Domain Agents (e.g., Growth, Tech, Legal)
- Red Teams (Compliance, Security, UX)
