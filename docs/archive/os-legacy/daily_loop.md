# Daily OS Loop

**Owner**: Daily Program Manager

## Cycle Checklist
1.  **Scan Signals**:
    - Backlog (Jira/GitHub)
    - Analytics (Conversion, Errors)
    - Synthetic User Reports
    - QA Failures
    - Research Queue (Knowledge gaps)
    - Trace Coverage (KPI)

2.  **Choose Bets**:
    - Select 1-3 Business Bets (Growth/Feature).
    - Select 1 Maintenance Bet (Tech Debt/refactor).

3.  **Execute (Parallel Pipelines)**:
    - Delegate to Agent Pods.
    - Agents use Skills to execute.
    - **Proof Requirement**: Must produce diffs, logs, screenshots.

4.  **Verify & Ship**:
    - QA Authority verifies DoD.
    - Compliance Gate checks Claims/Trace.
    - Ship via Feature Flag / Release Train.

5.  **Measure & Learn**:
    - Update Scoreboard.
    - Capture Lessons -> Update Skills/Policies.
    - Daily Report generation.
