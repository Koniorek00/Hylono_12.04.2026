# SAFETY RULES — ANTI-GRAVITY OS (PHASE 0)

1. **Secure Mode = ON**
   - Każda komenda terminala, skrypty przeglądarkowe i większe zmiany plików wymagają akceptacji użytkownika.
2. **Run-to-Completion Only**
   - Zakaz nieskończonych pętli autonomicznych. Misja -> Raport -> Stop.
3. **No Global Drift**
   - Agent nie dotyka plików poza rootem projektu (chyba że polecenie jest jawnie ręczne).
4. **Error-First Audit**
   - Każdy błąd narzędzia wymusza re-read `GEMINI.md` przed ponowną próbą.
5. **Truth-First Documentation**
   - Nie zmyślamy plików ani struktury. Jeśli czegoś nie ma, raportujemy brak.
