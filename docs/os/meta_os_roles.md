# Meta-OS (Control Plane) Roles

| Role | Responsibility |
| :--- | :--- |
| **Executive Orchestrator** | CEO/COO Brain. Prioritization, KPI ownership, conflict resolution, final decisions. |
| **DAILY Program Manager** | Autonomous daily runner. Selects bets, delegates work, produces daily reports. |
| **Org Evolution Steward** | Meta-level. Creates/retires agents, updates routing, prevents chaos. |
| **QA & Verification Authority** | Enforces Definition of Done (DoD). Veto power on releases without proof. |
| **Compliance & Safety Gate** | Global claim control. Veto on public claims/risky content. |
| **Security/Privacy Steward** | Privacy-by-design, secure mode hygiene. |
| **Incident Commander** | Ops. Handles outages, regressions, and postmortems. |
