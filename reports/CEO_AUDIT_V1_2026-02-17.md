## 🔴 CRITICAL — Fix immediately

### 1. Team Page Anonymity Undermines E-E-A-T
**Dimension:** Trust & Credibility  
**Impact:** Google's E-E-A-T guidelines require demonstrable expertise. Anonymous "Founder & CEO" titles hurt search rankings and customer trust.  
**Files:** [`components/AboutPage.tsx:9-24`](components/AboutPage.tsx:9)

```
→ content-product-writer: Create real team member profiles with credentials
→ visual-asset-director: Commission professional headshots
DO: Add real names, credentials (MD, PhD, etc.), LinkedIn profiles
DO NOT: Use placeholder text or stock photos
```

### 2. Missing hreflang Tags for EU Markets
**Dimension:** SEO & Discoverability  
**Impact:** Hylono targets EU customers but lacks hreflang implementation. This causes duplicate content issues and wrong locale ranking.  
**Files:** [`index.html:1-210`](index.html:1), [`components/SEO.tsx:1-150`](components/SEO.tsx:1)

```
→ i18n-specialist: Implement hreflang tags for en, pl, de minimum
DO: Add <link rel="alternate" hreflang="en" href="https://hylono.com/en/">
DO NOT: Skip this if targeting single market only
```

### 3. Server-Side Security Headers Unverified
**Dimension:** Security & Privacy  
**Impact:** CSP exists in meta tag but HSTS, X-Frame-Options, X-Content-Type-Options require server configuration. Without these, the site is vulnerable to clickjacking and MIME sniffing attacks.  
**Files:** [`index.html:9`](index.html:9), server configuration

```
→ security-compliance: Audit and implement security headers at CDN/hosting level
DO: Add Strict-Transport-Security: max-age=31536000; includeSubDomains
DO NOT: Rely solely on meta CSP for security
```
