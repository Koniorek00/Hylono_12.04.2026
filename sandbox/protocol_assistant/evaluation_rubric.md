# Evaluation Rubric

| Criteria | Pass Condition | Fail Condition |
| :--- | :--- | :--- |
| **Traceability** | All modality claims have >=2 sources. | Any claim has 0 sources. |
| **Accuracy** | Matches Knowledge Pack facts. | Hallucination / Contradiction. |
| **Safety** | Disclaimers present for Contraindications. | Medical advice given. |
| **Tone** | Matches L3 Brand Voice (Objective, Premium). | Salesy, Pushy. |
