## Hylono Conversion Hierarchy
Rental trial → Rental subscription → Purchase → B2B

**Always optimize for rental first.** It's lower friction and aligns with the mission.

## Friction Audit Framework

### Above-the-fold Test (10 seconds)
Can a visitor answer these in 10 seconds?
- [ ] What does Hylono sell?
- [ ] What problem does it solve?
- [ ] What does it cost / how do I try it?
- [ ] Can I trust this company?

### Friction Points by Stage

**Awareness → Interest**
- No clear category explanation (what IS mHBOT?)
- Too much jargon, not enough transformation language
- No social proof visible
- Hero takes too long to load

**Interest → Consideration**
- Pricing unclear or hidden
- Rental option buried below purchase
- No comparison tool (rent vs. buy)
- Too many CTAs competing for attention

**Consideration → Decision**
- Missing FAQ near CTA (objection handling)
- No trust signals near checkout button
- Deposit/commitment not clearly explained
- No "what happens next" after I click?

**Decision → Checkout**
- Too many form fields
- No guest checkout option
- No progress indicator in multi-step flow
- Address autocomplete not working

## CTA Hierarchy on Product Pages
```
Primary (rental): "Experience It — 30 days from €299/month"
Secondary (purchase): "Purchase" [smaller, below primary]
Tertiary (info): "Learn how it works" [text link]
```

## Trust Signals Near CTAs
Always place these within visual proximity of the primary CTA:
- Security badge / SSL indicator
- "No commitment" language for rentals
- Return/cancellation policy summary (1 line)
- Customer testimonial (1-2 sentences, attributed)
- "Free delivery and return collection" if applicable

## A/B Test Hypothesis Template
```
HYPOTHESIS: Changing [X] from [current] to [variation]
will increase [metric] because [psychological reason].

PRIMARY METRIC: [conversion rate / click rate / etc.]
SECONDARY METRIC: [bounce rate, time on page, etc.]
COMPLIANCE CHECK NEEDED: [Yes/No — if copy changes, always Yes]
MINIMUM SAMPLE SIZE: [calculate based on current traffic]
DURATION: [minimum 2 weeks, full business cycles]
```

## Objection Map (Hylono-specific)
| Objection | Response strategy |
|-----------|-----------------|
| "Is this safe?" | Safety section with contraindications, "Consult your provider" |
| "Does this actually work?" | Research references, user testimonials (compliant) |
| "Too expensive to buy" | Lead with rental price prominently |
| "What if I don't like it?" | 7-day trial option, easy return process |
| "I don't understand the technology" | Animated explainer, simple analogy |
| "Why Hylono vs. Amazon?" | Protocol + support + safety differentiator |

## Quick Win Checklist
- [ ] Rental CTA is the FIRST, LARGEST button on product pages
- [ ] Price visible without scrolling
- [ ] "No commitment" or equivalent near rental CTA
- [ ] FAQ section with 5 most common objections on every product page
- [ ] Social proof (testimonials) within 2 scrolls of fold
- [ ] Protocol explainer before purchase options
- [ ] Checkout form under 5 fields for initial step
